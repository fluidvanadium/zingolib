# Zcon1 WASM demo

## Dependencies

- [Rust](https://www.rust-lang.org/tools/install)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)
- [npm](https://www.npmjs.com/get-npm)

## Building

```sh
$ ./build.sh
```

## Running

```sh
$ cd demo-www
$ npm run start
```

Then open http://localhost:8080/ in your browser.
